Import / Export

📤 Dépenses — Excel

Export : sauvegarde le mois en cours. Import : colle des lignes Excel (une par ligne, tabulations) dans l'ordre Date · Catégorie · Commentaire · Montant. Tu peux aussi mettre une ligne « Nom de la catégorie » seule (ou suivie de tirets) au-dessus d'un bloc : elle s'appliquera à toutes les lignes suivantes qui n'ont pas de catégorie propre.

💰 Revenus — Excel

Colle un tableau par type de revenu : la 1re ligne donne le nom du type (ex. « Prestations informatiques »), puis une ligne par mois avec juste le montant reçu (« - » ou vide si aucun revenu ce mois-là). La cotisation est calculée automatiquement d'après le pourcentage réglé pour ce type dans Réglages. Une ligne « 2025 » seule change l'année des mois suivants (année en cours par défaut). Tu peux coller plusieurs types à la suite.
